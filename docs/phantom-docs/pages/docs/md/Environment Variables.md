### Assets

- `PHANTOM_ROOT_DIR` - Override phantom root directory, the default is `~/.phantom/`

### Updates

- `PHANTOM_NO_UPDATE_CHECK` - If set to any non-blank value, never check for updates

### Generate

#### Override All CC

- `PHANTOM_CC_64` - Override all paths to 64 bit cross compiler
- `PHANTOM_CC_32` - Override all paths to 32 bit cross compiler

#### Override Platform CC

- `PHANTOM_WINDOWS_CC_64` - Override path to Windows 64 bit cross compiler
- `PHANTOM_LINUX_CC_64` - Override path to Linux 64 bit cross compiler
- `PHANTOM_DARWIN_CC_64` - Override path to MacOS 64 bit cross compiler
- `PHANTOM_WINDOWS_CC_32` - Override path to Windows 32 bit cross compiler
- `PHANTOM_LINUX_CC_32` - Override path to Linux 32 bit cross compiler

#### Override All CXX

- `PHANTOM_CXX_64` - Override all paths to 64 bit cross compiler
- `PHANTOM_CXX_32` - Override all paths to 32 bit cross compiler

#### Override Platform CXX

- `PHANTOM_WINDOWS_CXX_64` - Override path to Windows 64 bit cross compiler
- `PHANTOM_LINUX_CXX_64` - Override path to Linux 64 bit cross compiler
- `PHANTOM_DARWIN_CXX_64` - Override path to MacOS 64 bit cross compiler
- `PHANTOM_WINDOWS_CXX_32` - Override path to Windows 32 bit cross compiler
- `PHANTOM_LINUX_CXX_32` - Override path to Linux 32 bit cross compiler
